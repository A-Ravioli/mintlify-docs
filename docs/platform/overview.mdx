## Dashboard

Access the Signal dashboard at `/signal` after logging in. The dashboard provides:

- **Real-time Metrics**: View credit balance, active runs, and spending
- **Quick Actions**: Create API keys, purchase credits, add integrations
- **Run Monitoring**: Track all your training runs in one place
- **History**: View past runs with detailed metrics and charts

## Architecture

```
Frontend (Next.js)
    ↓
Backend API (Port 8001)
    ↓
Supabase Database
    ↓
Signal API (Port 8000)
    ↓
Modal GPUs (Compute)
```

## User Flow

1. **Sign Up** → Create account and get initial access
2. **Purchase Credits** → Buy credits via Stripe
3. **Create API Key** → Generate keys for API access
4. **Configure Integrations** → Add WandB, HuggingFace, etc.
5. **Start Training** → Submit training jobs via API
6. **Monitor Progress** → Track runs in real-time
7. **Access Results** → Download checkpoints and metrics
